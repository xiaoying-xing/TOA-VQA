# TOA-VQA
Official code for "TOA: Task-oriented Active VQA" (Neurips 2023)

## Getting Started

### Dependencies
* python >= 3.6
* pytorch >= 1.7.1
* transformers
* h5py
  
### Executing program
Quick command to run the code
```
python main_vqa.py --filename name
```
